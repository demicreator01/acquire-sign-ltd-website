# Acquire Sign Ltd. - Luxury Landing Page

A mobile-first, dark cinematic luxury landing page for watches and jewellery.

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS v4

## Setup & Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design Notes
- Uses `Playfair Display` and `Raleway` via Google Fonts.
- Pure CSS/Tailwind animations (no external libraries).
- Smooth scrolling and "Ken Burns" effects.
- Image modal with zoom and pan.
